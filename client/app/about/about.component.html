Project Instruction
Challenge The aim of this challenge is to give us an idea of your level of expertise in Web Development. You are required to follow through with the two steps defined below:
  1. Build an API backend running on Node web server that have endpoints to handle a basic school information system. Make it limited to the following:
    • Creating Students’ resource
    • Reading Students’ resource
    • updating and deleting students
  2. By consuming the endpoints from the backend, build a frontend page that handles
    • Creating Students’ resource
    • Listing Students’ resource
    • Showing Students’ resource
    • editing and deleting students on backend
